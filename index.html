How to Play

Goal

Swipe across adjacent radical tiles to form valid Chinese characters.

Rules

  • Minimum 2 radicals - Each character needs at least 2 radicals
  • Adjacent only - You can only select adjacent tiles (including diagonals)
  • No reuse - Each tile can only be used once per character
  • Once per game - Each character can only be found once

Scoring

Points = sum of stroke counts of all component radicals

Example: 休 (人 + 木) = 2 + 4 = 6 points

Radicals

The game uses the 214 Kangxi radicals from the traditional Chinese dictionary system.
